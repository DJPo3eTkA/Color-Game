#Color Game Project

##Instructions:

--------------

-yarn install - download packages  
-yarn - run Gulp  
-yarn - build - final build  

Project based on  
- ES6 
- Pug 
- Sass  

--------------------------------------------------------------

It's recommended for you to install Node.js and Gulp globally.
